\doxysubsection{Подробное описание}
Класс для ведения логов в файл. 

Обеспечивает потокобезопасное логирование с возможностью указать файл для логов и несколькими уровнями серьезности сообщений. При невозможности записи в указанный файл, пытается создать файл \char`\"{}vcalc.\+log\char`\"{} в директории запуска.
